## 🚀 Overview

RoadCrashGuard is a **location-based road accident analysis system** that leverages geospatial data to identify accident patterns, high-risk zones, and provide personalized safety recommendations.

The system fetches **user location or manually searched locations**, analyzes accident data, and generates meaningful insights to help users make safer travel decisions.

---

## ✨ Key Features

* 📍 Fetch live / user-input location
* 🗺️ Location-based accident data analysis
* 🔍 Identify high-risk accident zones
* 📊 Data preprocessing & pattern detection
* 📈 Trend analysis & insights generation
* 🌐 Interactive maps using Folium
* ⚡ Real-time safety recommendations
