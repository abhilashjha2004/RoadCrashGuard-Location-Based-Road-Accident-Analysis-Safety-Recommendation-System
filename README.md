# 🚦 RoadCrashGuard – Location-Based Road Accident Analysis & Safety Recommendation System

RoadCrashGuard is a data-driven system that analyzes road accident data and identifies dangerous locations (hotspots).  
It provides safety insights and recommendations using data analytics and interactive visualizations.

---

## 📌 Features

- 📊 Automatic Data Analysis
- 🗺️ Interactive Accident Hotspot Map
- 📈 Safety Score Calculation
- 📍 Location-Based Risk Detection
- 🔍 Historical Accident Analysis
- 📉 Data Visualization using Plotly
- 🧠 Intelligent Safety Recommendation Engine

---

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Folium
- DBSCAN Clustering
